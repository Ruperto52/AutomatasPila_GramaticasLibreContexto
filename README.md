# AutomatasPila_GramaticasLibreContexto
Dentro de este proyecto se abordan lo que son los automatas de pila (AP), como funcionan y para que nos sirven. Al igual que la construccion de las gramaticas libres de contexto (GLC), que son, como se contruyen y finalmente la conversion entre estos (AP -> GLC) y viceversa (GLC ->AP)
